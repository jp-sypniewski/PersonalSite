Welcome to JP's (not so very pretty) Personal Site!

As an analyst at Dish Network (~2018/9), I began to dabble in programming. My first memorable experience was in Python, through Al Sweigart's Automate the Boring Stuff with Python. I was able to wrangle a massive overload of team files on a shared drive, handle more data than my peers using Excel, and open up the piloting of agent workflows (previously unaccessible due to 'too much work').

In November 2019, I began the Skill Distillery Full Stack Web Development Bootcamp. The course is split into three sections:

  • Java and fundamentals of programming
  • Web frameworks and data persistence [MySQL, JDBC, JPA, Spring Boot]
  • Web application technoligies [Javascript, Angular]

After the Java section, the course covered and provided experience in many topics in quick succession, including:

Tools Technologies, Methodologies Concepts
Spring Tool Suite, Postman Model-View-Controller Web Framework, RESTful API, Object Relational Mapping JDBC, JPA
MySQL Workbench, MAMP SQL, MySQL Referential Integrity, Prepared Statements to avoid SQL Injection
